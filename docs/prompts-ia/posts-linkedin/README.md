# Posts LinkedIn — Prompts para Publicações

> Prompts para gerar posts de LinkedIn sobre o projeto.
> Objetivo A (vagas) ou B (conhecimento).

---

## Arquivos

| Arquivo                                      | Uso                                                                             |
| -------------------------------------------- | ------------------------------------------------------------------------------- |
| [PROMPT-LINKEDIN.md](./PROMPT-LINKEDIN.md)   | Prompt completo. Copie o bloco e cole no chat da IA.                            |
| [CONTEXTO-PROJETO.md](./CONTEXTO-PROJETO.md) | Resumo do projeto (implementações, ferramentas, guias) para contexto adicional. |

---

## Uso rápido

1. Copie o bloco em [PROMPT-LINKEDIN.md](./PROMPT-LINKEDIN.md).
2. Cole no chat da IA.
3. Peça: _"Gera um post sobre [tema], objetivo A"_ (vagas) ou _"objetivo B"_ (conhecimento).
4. A IA gera o post ajustado.

Se não informar A ou B, a IA pergunta antes de gerar.
