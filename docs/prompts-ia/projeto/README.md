# Prompt Projeto — Regras Completas para IA

> Prompt consolidado com todas as regras, convenções e fluxos do Dev Showcase.
> Use quando a IA for criar ou modificar código no projeto.

---

## Uso

1. Copie o bloco em [PROMPT-PROJETO.md](./PROMPT-PROJETO.md).
2. Cole no início da conversa com a IA (Cursor, ChatGPT, Claude).
3. Em seguida, peça a tarefa: _"Cria uma nova página de guia sobre X"_, _"Adiciona um componente Y"_, etc.

A IA terá todas as regras carregadas e seguirá as convenções do projeto.

---

## Quando usar

- Criar nova página (guia, implementação, ferramenta)
- Adicionar componente
- Criar nova API route
- Adicionar traduções
- Refatorar código existente
- Qualquer alteração que precise seguir os padrões do projeto
